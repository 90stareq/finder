Welcome!

WeCanHearYou is an open source, cross platform and free to use feedback system for companies and products.

Current stage

We're currently bootstraping both the application, so there isn’t much to see. You can keep track of its progress using our GitHub repositories.

Features

  • One click log in with OAuth and Social Login
  • Cloud or Self hosted. It's up to you
  • Let your users shares ideas on how to make your project better
  • The community can vote and improve existing ideas
  • Organize ideas using tags
  • Share private ideas within your team
  • Lightning fast. Because yes, performance is a feature

Live environment

Our staging environment is hosted by Heroku and it’s automatically deployed upon commit on dev/master.

* Demo: http://orange.staging.canhearyou.com

How to install it

No instructions are available yet, but you'll need:

* Linux/Windows server. There is no runtime dependency.

* A PostgreSQL database.

We plan to add Docker support on a later stage to make self-hosted deployment a lot easier.
