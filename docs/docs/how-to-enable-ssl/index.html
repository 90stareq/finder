How to enable SSL

Follow this quick tutorial to know how you can host your instance under the HTTPS protocol.

Prerequisites

In order to configure HTTPS, you’ll need a Certificate and the Private Key.

How to enable it

Fider v0.3.0 shipped with two new environment variables that you’ll need to set.

  1. SSL_CERT: This is the path to the Certificate file.
  2. SSL_CERT_KEY: This is the path to the Private Key file.

If you’re using Docker, these files won’t be inside the container, but Fider will need to have access to it. In order to accomplish this, we recommend the use of data volumes.
