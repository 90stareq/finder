How to enable SSL

Host your instance under the HTTPS protocol

Prerequisites

In order to configure HTTPS, you’ll need a Certificate and the Private Key.

How to enable it

These are the two environment variables that have to be set.

  1. SSL_CERT: The name of the Certificate file.
  2. SSL_CERT_KEY: The name of the Private Key file.

Fider’s Docker image expects both file to be available on /app/etc. You can copy these files anywhere you want on your server and share them with your Docker container by using volumes.

E.g.: if both files are on /home/me/fider, you can map it to the container with -v /home/me/fider:/app/etc.
